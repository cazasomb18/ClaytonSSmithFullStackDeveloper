Scintillating Sequences is 2 player memory game built with HTML5, jQuery, JS and CSS. Each player takes turns using keystroke to replicate a randomized pattern that appears on the UI. The longer you last the harder it gets...

Scintillating Sequences

ToolShed is a full stack application built using Express, Node, and EJS pages. The aim of the project was to gain confidence with RESTful routing while delivering CRUD capability. This crowd sourced platform enables users to Create and account, load photos, and post descriptions of their tools available for loan. Users an also post comments on other posted tools as well as edit and delete their own comments and/or tool posting(s). The function of ToolShed is to create a crowd-sourced solution for lending/borrowing power tools geared towards those living in urban areas.

ToolShed

Late Night Bytes is a MERN stack fully decoupled application which consumes 2 APIs, implements geolocation based search results and presents full crud functionality to users. Users who login are automatically shown a list of nearest "late night restaurants" as well as a marker placed on the google maps feature for each respective search result. You can even tell how close you are, exactly, to the restaurants returned from the google-places API query due to the different markers.
