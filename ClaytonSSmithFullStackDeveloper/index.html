Scintillating Sequences

  • Scintillating Sequences is a 2 player memory game built with HTML5, jQuery, JavaScript and CSS3
  • Each player takes turns using keystrokes to replicate a randomized pattern that appears on the UI
  • The more rounds you complete the harder it becomes to repeat the sequence
