Scintillating Sequences

  • Scintillating Sequences is a 2 player memory game built with HTML5, jQuery, JavaScript and CSS3

  • Each player takes turns using keystrokes to replicate a randomized pattern that appears on the UI

  • The more rounds you complete the harder it gets to complete the sequence

ToolShed

  • ToolShed is a full stack application built using Mongoose, Express, Node, and EJS pages
  • Enables users to create an account, load photos and post descriptions of their tools available for loan. Users can also post comments on other posts and edit/delete their own comments/posts
  • The goal of ToolShed is to create a crowd-sourced solution for ubran dwellers looking to lend and borrow power tools

Late Night Bytes

  • Late Night Bytes is a MERN stack fully decoupled application which consumes 2 APIs, implements geolocation based search results and presents full CRUD functionality to users
  • Users who login are automatically shown a list of nearest "late night restaurants" based on their current location
  • Users can tell how close they are to the restaurants due to the different marker icons
