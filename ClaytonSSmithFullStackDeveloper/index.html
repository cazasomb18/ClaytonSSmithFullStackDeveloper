Scintillating Sequences

  • Scintillating Sequences is a 2 player memory game built with HTML5, jQuery, JavaScript and CSS3

  • Each player takes turns using keystrokes to replicate a randomized pattern that appears on the UI.

  • The more rounds you complete the harder it gets to complete the sequence.

ToolShed

  • ToolShed is a full stack application built using Mongoose, Express, Node, and EJS pages. The aim of the project was to gain confidence with RESTful routing while delivering CRUD capability.
  • This app enables users to create an account, load photos, and post descriptions of their tools available for loan. Users can also post comments on other posts as well as edit/delete their own comments/posting(s).
  • The goal of ToolShed is to create a crowd-sourced solution for lending and borrowing power tools geared towards those living in urban areas.

Late Night Bytes

  • Late Night Bytes is a MERN stack fully decoupled application which consumes 2 APIs, implements geolocation based search results and presents full CRUD functionality to users.
  • Users who login are automatically shown a list of nearest "late night restaurants" as well as a marker placed on the google maps feature for each respective search result.
  • You can even tell how close you are to the restaurants returned from the google-places API query due to the different marker icons.
